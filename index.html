Verificación
Geometría conocida,
¿cumple o no cumple?
Ingresas las dimensiones (B, L, Df), las cargas por columna y el perfil estratigráfico. STRATEX calcula la capacidad portante por Meyerhof, las presiones en profundidad por Boussinesq y los asentamientos totales. El diagnóstico NSR-10 es automático.
  • M1–M2: geometría de zapatas y perfil de suelo
  • M3–M4: presiones efectivas y Meyerhof completo
  • M5–M8: Boussinesq, asentamientos y diferencial
  • M9–M11: visualizaciones, reporte y vista 3D
⚙️
Diseño
BETA
Cargas y criterios dados,
STRATEX propone el tamaño
Defines las cargas por columna, el factor de seguridad objetivo (FS) y el asentamiento admisible. STRATEX itera Df y B automáticamente para encontrar la zapata de menor volumen que satisfaga ambos criterios, y luego ejecuta la verificación completa.
  • Optimización discreta: barrido Df 0.5–3.0 m × B 0.5–10 m
  • Restricción doble: FS ≥ objetivo y S ≤ Sadm
  • Criterio de mínimo volumen entre todas las soluciones válidas
  • Verificación completa automática al terminar el diseño

Arquitectura del software

11 módulos integrados,
flujo de cálculo encadenado

Cada módulo implementa sus ecuaciones en JavaScript puro mediante fórmulas cerradas. Los resultados se propagan automáticamente entre módulos al presionar "Calcular Todo".

M1
Geometría de Zapatas
B, L, Df, carga Q, momentos Mx/My, tipo (cuadrada/rectangular/corrida), ángulo β, posición X–Y en planta
M2
Perfil Estratigráfico
N estratos: c, φ, γ, γsat, Es, μ, e₀, Cc, Cs, σ'p, cv, USCS. Nivel freático Nf. Múltiples sondeos con coordenadas
M3
Presiones Efectivas
σ'vo(z) por integración acumulada capa a capa. Corrección automática γ' = γsat − γw bajo el nivel freático
M4
Meyerhof Completo
Nc, Nq, Nγ + 9 factores (forma, profundidad, inclinación). Parámetros ponderados en zona de falla [Df, Df+B]. 3 casos freático
M5
Boussinesq Analítico
Δσz(z) por fórmula cerrada rectangular con superposición de 4 esquinas. Manejo del caso arctan con argumento negativo
M6
Asentamiento Granular
Schmertmann-Hartman (1978): perfil Iz(z) analítico, factor C1 de alivio de esfuerzos, factor C2 de fluencia en t años
M7
Asentamiento Cohesivo
Timoshenko-Janbu + Is de Giroud (1968) para Si inmediato. Consolidación Terzaghi NC/OC casos A y B con Δσz de Boussinesq
M8
Asentamiento Diferencial
δ = |Si − Sj| entre todos los pares de zapatas. Distorsión β = δ/L. Semáforo automático NSR-10 H.4.2: β ≤ 1/300
M9
Visualizaciones
Perfil estratigráfico con textura · planta de cimentación · diagrama Δσz vs z · mapa de calor de asentamientos (Chart.js)
M10
Reporte de Cálculo
Memoria completa exportable a PDF y Excel. Incluye todos los valores intermedios: factores BCF, 9 factores Meyerhof, Δσz, asentamientos diferenciados por método
M11
BETA

Modelo Three.js con estratos a escala real, zapatas a su Df, animación de asentamiento progresivo y tres modos: Mapa Δσz, Bulbos σz y Distribución de presiones
Implementación matemática

Fórmulas cerradas, ningún ábaco

Todas las ecuaciones están programadas explícitamente. Ningún resultado depende de un valor interpolado de una tabla o leído de un ábaco.

// Factores de capacidad de carga — Meyerhof (1963)
const Nq = φ => Math.exp(Math.PI * Math.tan(φ)) * Math.tan(Math.PI/4 + φ/2)**2;
const Nc = φ => φ < 1e-5 ? 5.14 : (Nq(φ)-1) / Math.tan(φ);
const Nγ = φ => (Nq(φ)-1) * Math.tan(1.4*φ);

// Boussinesq — fórmula cerrada rectangular (superposición por 4 esquinas)
function boussCorner(m, n) {
  const mn2 = m*m+n*n, A = 2*m*n*Math.sqrt(mn2+1) / (mn2+m*m*n*n+1);
  return (1/(4*Math.PI)) * (A*(mn2+2)/(mn2+1) + Math.atan(A/(1-m*m*n*n/(mn2+1))));
}

// Giroud (1968) — Is analítico para cohesivos, sin tabla, sin ábaco
const giroudIs = (B,L) => {
  const m=L/B;
  return (2/Math.PI)*(Math.log(m+Math.sqrt(m*m+1))+m*Math.log((1+Math.sqrt(m*m+1))/m));
};

// Consolidación NC/OC (Terzaghi) con Δσz de Boussinesq estrato a estrato
// Caso NC:    Sc = Cc·H/(1+e₀) · log₁₀((σ'v0+Δσ) / σ'v0)
// Caso OC→A:  Sc = Cs·H/(1+e₀) · log₁₀((σ'v0+Δσ) / σ'v0)  [σ'v0+Δσ ≤ σ'p]
// Caso OC→B:  combinación Cs hasta σ'p + Cc más allá — punto de quiebre analítico
Características del software

Motor de cálculo completo,
interfaz de ingeniería

⚙️
Modo Diseño con optimización automática
Dado un conjunto de cargas y criterios (FS, Sadm), STRATEX itera sobre Df y B para encontrar la zapata de mínimo volumen que satisface capacidad portante y asentamientos simultáneamente.
🧮
Selección automática de método de asentamiento
Identifica si cada estrato en la zona de influencia es granular (→ Schmertmann-Hartman) o cohesivo (→ Timoshenko-Janbu + consolidación), sin intervención del usuario.
📍
Asignación sondeo–zapata por proximidad
Con coordenadas X–Y en los sondeos, STRATEX asigna automáticamente el perfil de suelo más cercano a cada zapata. Sin coordenadas, usa el sondeo más desfavorable.
📐
3 casos de nivel freático (Meyerhof)
Corrección automática de γ* según posición del NF relativa a Df, Df+B o fuera de la zona de influencia. Se aplica al término Nγ con interpolación lineal.
✅
Verificación NSR-10 completa
FS ≥ 3, S ≤ 25 mm y β ≤ 1/300 verificados para cada zapata y cada par de zapatas. Diagnóstico visual con semáforo y márgenes porcentuales en M4, M7 y M8.
🎮
Vista 3D animada (Three.js)
Modelo interactivo con estratos a escala real, zapatas en su posición X–Y a su Df correcto, simulación animada del asentamiento progresivo y controles de órbita completos.

Cimentaciones Profundas

La siguiente versión extiende STRATEX al análisis y diseño de sistemas de pilotaje. Misma filosofía: fórmulas cerradas, flujo automático, sin ábacos.

Pilotes Individuales
Análisis de capacidad axial de pilotes aislados en suelos estratificados. Resistencia por fuste y resistencia de punta calculadas estrato a estrato.
  • Capacidad en fuste: adhesión y fricción por tipo de suelo
  • Capacidad en punta: Meyerhof y Vesic para pilotes
  • Asentamiento de pilote individual en suelo estratificado
  • Factor de seguridad y carga admisible axial
Disponible en v2.0
🔒 🏗️
Grupos de Pilotes
Análisis de eficiencia y capacidad de grupos de pilotes bajo cargas combinadas. Distribución de cargas entre pilotes y verificación del grupo como bloque.
  • Eficiencia de grupo: método de Converse-Labarre y Feld
  • Distribución de cargas axiales y momentos en la cabeza
  • Verificación del grupo como zapata equivalente profunda
  • Asentamiento del grupo por compresión del bloque de suelo
